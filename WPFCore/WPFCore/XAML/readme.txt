AMRisk.XAML
===========

Classes in this namespace provided helper used by XAML.

Note: the folder (Internal) contains - as the name suggests - internally used resources. Items in this folder
      do not extend the namespace by the folder name, i.e. they are all located in the AMRisk.XAML namepsace.


ListViewExtensions:
- CopyToClipboard: retrieves the values from an ItemsControl and places them in the clipboard (useful in conjunction with a ListView)
